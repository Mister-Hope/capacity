# 教案：电容器的电容（第一课时）

| 项目 | 内容 |

| 教材 | 人教版 必修三 第十章 第4节 |
| 课时 | 第1课时（共2课时），45分钟 |
| 授课教师 | 张伯望 |

## 教学目标

### 知识与技能
1. 理解电容器的定义：相互靠近且彼此绝缘的两个导体构成电容器
2. 能够描述电容器的充电和放电过程
3. 理解电容器带电量为一个极板带电的绝对值
4. 知道充放电过程中的能量转化

### 过程与方法
1. 通过类比和发散思维，理解电容器定义的广泛适用性
2. 通过分析充电/放电过程，建立微观电荷运动的物理模型

### 情感态度与价值观
1. 体会"储存电荷"这一工程需求背后的物理思想
2. 认识电荷守恒定律的普遍性

## 教学重难点

- **重点**：电容器的定义，充电和放电过程
- **难点**：电容器保持电荷的原理（静电引力的作用）

## 教学过程

### 一、引入（约2分钟）

提出问题"如何把电荷存起来？"回顾已学的三种起电方式，指出它们只能让物体带电，不能储存电荷，自然引出电容器。

### 二、电容器的定义（约3分钟）

给出定义，强调"相互靠近"和"彼此绝缘"两个条件。发散举例：地球-火星、人-金属板、云层-大地均可视为电容器。

### 三、常见电容器（约1分钟）

展示桶状电容器、陶瓷电容器、铝电解电容器的实物图片，建立感性认识。

### 四、充放电过程（约4分钟）

分别讲解充电和放电的微观过程：电子如何在两极板之间转移。强调电荷守恒。

### 五、保持电荷的原理（约3分钟）

通过"负极接地"的思想实验，说明静电引力使得正极板电荷不会流失。这是电容器设计的核心思想。

### 六、带电量的定义（约2分钟）

基于电荷守恒，明确电容器带电量为一个极板带电的绝对值 Q。

### 七、能量转化（约3分钟）

分析充放电过程中的能量转化：充电时电源能量储存于电容器，放电时释放为其他形式能量。总结电容器是电场储能元件。

### 八、课堂小结（约2分钟）

回顾电容器定义、充放电过程、能量转化三条主线。

## 板书设计

```
电容器的电容

一、定义：相互靠近 + 彼此绝缘 → 储存电荷
二、充放电：
    充电：电源 → 电子转移 → 两极带电
    放电：灯泡 → 电子回流 → 电荷中和
三、带电量：一个极板带电的绝对值 Q
四、能量：电容器 = 电场储能元件
```
