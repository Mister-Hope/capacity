## 教学目标

### 知识与技能

1. 理解电容器的定义：相互靠近且彼此绝缘的两个导体构成电容器，能够储存电荷
2. 能够描述电容器的充电和放电过程，知道电子在回路中转移的方向
3. 理解电容器带电量为一个极板带电的绝对值 Q
4. 知道充放电过程中伴随着能量转化

### 过程与方法

1. 通过发散思维（地球-火星、人-金属板、云层-大地），理解电容器定义的广泛适用性
2. 通过交互式电路模拟，观察充放电过程中电子流动、极板带电和电势差变化

### 情感态度与价值观

1. 体会"储存电荷"这一工程需求背后的物理思想
2. 认识电荷守恒定律在电容器问题中的体现

## 教学重难点

- **重点**：电容器的定义，充电和放电过程的微观解释
- **难点**：电容器保持电荷的原理（静电引力 + 绝缘条件）

## 教学过程

### 一、引入：如何储存电荷？（约2分钟）

提出问题"我们如何把电荷存起来？"回顾三种起电方式（摩擦起电、接触起电、感应起电），指出它们只能让物体带电，不能储存电荷，自然引出电容器。

### 二、电容器的定义（约2分钟）

给出定义："相互靠近且彼此绝缘的两个导体，构成电容器。"分别阐述"相互靠近"（电荷相互吸引，不会流失）和"彼此绝缘"（没有导电通路，电荷无法逃逸）两个条件。

### 三、发散举例（约1.5分钟）

将电容器概念推广到生活中的例子：地球与火星（真空绝缘）、人与金属板（空气绝缘，不接触）、云层与大地（云层带电，大地感应）。总结：任意两个导体，靠近+绝缘即可视为电容器。

### 四、常见电容器（约1分钟）

展示陶瓷电容器（固定电容，电路板常用）、铝电解电容器（大容量，有极性）、可变电容器（旋转调节，改变正对面积）三类实物照片。

### 五、充放电交互演示（约2分钟）

使用交互式电路组件演示：单刀双掷开关切换充电与放电。充电时电流表读数由大变小，极板由灰变红（+）蓝（-）；放电时短接，电流反向，极板颜色恢复。同时展示 I-t 暂态电流曲线。

### 六、充放电过程解析（约2分钟）

**充电**：与正极相连的极板失去电子→带正电；与负极相连的极板获得电子→带负电；两极板间电势差逐渐升高至等于电源电压；电源的能量储存在电容器中。

**放电**：用导线连接两极板，电子从负极板经导线流向正极板；两极板电荷逐渐中和，电势差降低；电容器储存的能量释放到电路中。

总结：电荷守恒——充电不是创造电荷，放电不是消灭电荷，只是转移。

### 七、为什么能保持电荷？（约2分钟）

抛出问题：充好电的电容器，负极接地，电荷会流失吗？→ 不会。因为负极板上的负电荷与正极板上的正电荷相互吸引，正极板电荷被"拉住"。扩展：即使接上导线和电阻，只要不构成闭合回路，电荷仍大部分保留。结论：相互靠近+彼此绝缘→有效储存电荷。

### 八、电容器的带电量（约1.5分钟）

问题引入：充放电是电荷在两极板之间转移，那么电容器的带电量如何计算？由电荷守恒知：当一个极板带电量为 +Q 时，另一个极板必然带电 −Q。电容器的带电量 = 一个极板带电的绝对值 Q。

### 九、充放电中的能量转化（约2分钟）

**充电**：两极板间电势差逐渐增大，电源的能量→储存在电容器中。**放电**：两极板间电势差逐渐减小，电容器储存的能量→释放到电路中。总结：电容器不仅可以储存电荷，还同时存储了能量。

### 十、探究：不同电容器的储存本领（约2分钟）

抛出问题"不同电容器的储存本领是否一样？如何量化？"，自然过渡到定量探究。

### 十一、电荷共享实验（约3分钟）

使用交互式电路组件演示：S₁接1给C₁充电至8V → S₁接2与C₂并联 → 电压表示数变为一半。闭合S₂放电后重复，可精确验证Q∝U。体现"先问后答"——实验结论分两步呈现。

### 十二、实验结论（约1分钟）

两个列表项：(1)同一电容器，Q与U比值是定值；(2)不同电容器，该比值不同。

### 十三、水桶类比（约1.5分钟）

水桶横截面积S=V/h ←→ 电容C=Q/U。水龙头注水到出水口即停 ←→ 电容器充电到电源电压即停（后续做动效）。

### 十四、电容定义（约2分钟）

三步递进：一、定义（Q与U的比值，反映储存本领）；二、定义式C=Q/U；三、单位——法拉，简称法，符号F，1F=1C/V，常用μF、pF。

### 十五、电容器上标的电压（约2分钟）

聊天气泡互动形式：问"电容器上标的电压是什么？"→答"额定工作电压"。问"闪电给了什么启示？"→答"电势差过大→空气击穿→闪电放电"。

### 十六、工作电压与击穿电压（约1.5分钟）

两点：额定工作电压（长期可靠工作的最大电压）、击穿电压（介质击穿→导电通路→不可逆损坏）。安全裕度表述并入击穿电压段。

### 十七、转折：电容和什么有关？（约1分钟）

灯泡图标 + 提问："电容——电容器的储存本领——和什么有关？以最简单的平行板电容器为例，展开小组讨论。"右下角复用封面电容器SVG装饰。

### 十八、平行板电容器（约3分钟）

交互式组件：三滑块（S:100~500cm², d:3~8mm, U:10~50V）+ 介质切换（真空/玻璃/陶瓷）。调节S↑→C↑，d↑→C↓，U变化→Q等比例变化但C不变。结论：C由S、d、ε_r决定，与U无关。

### 十九、课堂小结（约1分钟）

四块：电容器定义（相互靠近+彼此绝缘→储存电荷）、电容C=Q/U（单位法拉）、平行板电容器C=ε_rS/(4πkd)（S↑C↑, d↑C↓）、工作电压提醒（底部小字）。

## 板书设计

```
电容器的电容

一、定义：相互靠近 + 彼此绝缘 → 储存电荷的容器
二、充放电：
    充电：电子转移 → 两极带电 → 电势差升高 → 电能储存
    放电：电子回流 → 电荷中和 → 电势差降低 → 能量释放
三、带电量：一个极板带电的绝对值 Q（电荷守恒）
四、电容：C = Q/U，单位法拉(F)，1F = 1C/V
五、平行板电容器：C = ε_r S / (4π k d)
    S↑ C↑ | d↑ C↓ | ε_r↑ C↑
六、工作电压：额定 < 击穿，留安全裕度
```
