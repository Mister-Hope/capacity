# 高中物理交互式课件

## 项目概述

面向大教室公开课的高中物理交互式课件，基于 Slidev + Vue 构建。内容涵盖运动学、力学、电磁学、波动光学等高中物理核心章节。

## 设计原则

### 核心理念：博物馆级信息设计

每一页幻灯片都应该像一件设计作品——清晰、优雅、有呼吸感。不仅仅是教学工具，更是视觉享受。

### 视觉规范

- **大圆角**：所有卡片、容器使用 `border-radius: 1.25rem ~ 2rem`，营造柔和、现代的视觉感受
- **留白优先**：内容密度低，每页聚焦一个核心概念，大量留白让信息呼吸
- **字号**：公开课大教室专用，基础字号 28px+，标题 48-72px，确保最后一排清晰可读
- **配色**：深蓝学术基调（`#1a1a2e` 背景），点缀暖金色（`#e8b86d`）和亮白（`#fafafa`），物理量用青绿色（`#2ec4b6`）
- **卡片风**：关键内容使用带阴影的圆角卡片承载
- **统一过渡**：页面切换使用 `fade` 过渡，全局层保持标题和进度条在页面间连续

### 动画与过渡

- 页面间过渡使用 `transition: fade`，配合 `global-top.vue` / `global-bottom.vue` 保持标题、进度条等元素在页面切换时的连续性
- 页面内使用 `v-click` 控制内容逐步呈现
- 交互式物理模拟组件使用 Vue 响应式驱动 Canvas/SVG 动画
- 避免花哨动画，所有动效服务于教学内容

### 技术约定

- 包管理：`pnpm`（不要用 npm 或 yarn）
- LaTeX：Slidev 内置 KaTeX，行内 `$...$`，行间 `$$...$$`
- 组件：Vue 3 组件放在 `components/` 目录，Slidev 自动注册
- 全局层：`global-top.vue`、`global-bottom.vue` 放在项目根目录
- 样式：全局样式在 `style.css`，组件内使用 `<style scoped>`
- Mermaid 图表用于受力分析等流程图
- 交互模拟组件使用 Canvas API 或纯 SVG

### 课件结构

```
slides.md          ─ 主课件入口
components/        ─ 交互式 Vue 组件（自动注册）
global-top.vue     ─ 全局顶部栏（章节标题 + 进度）
global-bottom.vue  ─ 全局底部栏（学校 Logo + 页码）
style.css          ─ 全局样式系统
layouts/           ─ 自定义布局（可选）
```
